VPM レポジトリのマニフェストを GitHub レポジトリから生成

GitHub レポジトリ パッケージ ID package.json のパス

開発者は…

パッケージのマニフェスト package.json からレポジトリを自動生成する API により、レポジトリ個別の設定が必要なくなります。 レポジトリに release と対応する zip ファイルを追加すると自動で反映されるため、CI の設定も必要ありません。 1 GitHub レポジトリに 1 VPM レポジトリが対応するため、シンプルな構成が可能です。 デフォルトで io.github.owner.repo のパッケージ ID が割り当てられますが、ほかの ID を指定することも可能です。

ユーザーは…

1 GitHub レポジトリに 1 VPM レポジトリが対応するため、望んだパッケージのみをインストールできます。 ただし、前提パッケージを自分で読み込む必要があります。
